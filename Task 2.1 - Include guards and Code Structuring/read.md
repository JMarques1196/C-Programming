# Objectives:
* Learn the importance of header guards.
* Use #ifndef, #define #endif correctly
* Create header files with function declarations
* Organize code respecting the separation of concerns

# Task:
* Create a calculator.h file:
    * Must contain appropriate header guards
    * Use the following functions:
        * int multiply(int a, int b)
        * int subtract(int a, int b)

* Make sure the identifier of the header guard is unique
* Make sure the file contains only prototypes / declarations
* Mantains a consistent format and adequate comments
