# Theme:
Build a complete project, separating the interface from the implementation. Emphasis on modulation and best practices integrating multiple files.

# Competences:
* Develop a modular project with multiple files
* Separate the declarations (.h) from the definitions (.c)
* Apply coding best practices in organizing code, documentations and comments
* Compile and link multiple files
* Test and validate module integration

# Task:
* Create a small library containg geometric functions using 3 files:

## geo.h:
    * Implement header guards
    * Declare functions with comments explaining them.

## geo.c:
    * Implement the functions declared in geo.h
    * Use n = 3.14159 for calculations
    * Include the header file
    * Comment the code, using a clear language

## test.c
    * main functions that tests the 3 files.
